# CyberGuard AI - Digital Immunity Platform

AI-powered cyber safety training and vulnerability assessment platform.

## 🔥 Core Features
- **Real-World Scam Simulations**: Controlled, safe phishing tests to measure user behavior.
- **Behavioral Analysis**: Detects psychological triggers like urgency, fear, and authority bias.
- **Dynamic Risk Score**: A real-time 0-100 score indicating digital vulnerability.
- **AI Scam Checker**: Paste suspicious messages to analyze their threat level.
- **Adaptive Training**: Difficulty increases as the user's digital immunity strengthens.

## 🏗 Tech Stack
- **Frontend**: Vite + React + Framer Motion (Animations) + Lucide Icons
- **Design**: Modern Glassmorphism with Neon Teal & Deep Purple palette.
- **Backend (Future)**: FastAPI + PostgreSQL + ML Risk Engine.

## 🚀 Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   npm run dev
   ```

## 📊 Methodology
The system calculates a **Cyber Risk Score** based on:
- Click failure rate
- Response time
- Reporting behavior
- Scam detection accuracy
